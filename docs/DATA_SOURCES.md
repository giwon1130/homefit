# Data Sources — homefit

외부에서 가져오는 데이터와 연동 방식. 유료/무료, Rate limit, 이용약관 핵심을 함께 기록한다.

## 1. 청약 공고

### 1.1 공공데이터포털 — 한국부동산원 청약홈 분양정보
- **기관:** 한국부동산원
- **API 이름:** 청약홈 분양정보 (서비스키 발급 후 사용)
- **형식:** REST, JSON/XML
- **비용:** 무료 (활용신청 승인 필요)
- **업데이트:** 공고 등록 시점 기준 거의 실시간
- **주의:** 일 트래픽 제한 존재 → 일 1~2회 전체 싱크 + 상세는 온디맨드

### 1.2 LH 청약플러스
- 공식 오픈 API는 제한적. 공공데이터포털의 "LH 임대주택 공고" API 확인
- 추가로 LH 보도자료 RSS + HTML 크롤링 폴백

### 1.3 SH공사, GH공사, 각 지방 도시개발공사
- 별도 공공데이터셋 탐색 (데이터셋 검색 키워드: "매입임대", "전세임대", "행복주택")
- API 없으면 웹 크롤링 (robots.txt 준수)

## 2. 실거래가

### 2.1 국토교통부 실거래가 공개시스템
- **API:** 아파트 매매 실거래가 상세자료, 아파트 전월세 자료
- **형식:** REST, XML (일부 JSON 지원)
- **비용:** 무료
- **파라미터:** 법정동코드(5자리 시군구) + 계약년월(YYYYMM)
- **전략:** 전국 시군구 × 월 단위로 초기 백필 후 일일 증분

## 3. 지도 / 경로

### 3.1 카카오맵
- **JS SDK** (웹), **iOS/Android SDK** (모바일)
- **지오코딩 REST API** (주소 → 좌표)
- **비용:** 일정 트래픽 무료, 이후 과금
- **키 관리:** 웹용/모바일용/REST용 키 분리, 도메인/번들ID 제한 설정

### 3.2 카카오모빌리티 길찾기 API
- **용도:** 자동차 경로
- **비용:** 유료 (베타/파트너)

### 3.3 ODsay 대중교통
- **용도:** 지하철/버스 환승 경로
- **비용:** 무료 티어 존재 (일정 호출수 제한)
- **전략:** 통근 계산의 핵심. 결과 **반드시 캐시** (30일 TTL)

### 3.4 VWorld 지오코딩 (폴백)
- 카카오가 실패했을 때 백업

## 4. 주소 / 행정구역

- **행정안전부 법정동 코드**: 공공데이터포털에서 CSV 다운로드 후 `admin_dong` 정적 테이블로 적재
- **우편번호**: 필요 시 도로명주소 API (추후)

## 5. 이용약관 / 법적 주의

- 청약홈, LH 등 공공 사이트는 **크롤링 허용 여부를 robots.txt**에서 매번 확인
- 공식 API 우선, 크롤링은 폴백으로만
- User-Agent 명시: `homefit-ingestion/{version} (contact: <email>)`
- 호출 간격: 최소 1초
- 캐시: 공개 데이터라도 **원본 공고 URL은 반드시 함께 저장**해서 사용자가 1차 출처를 볼 수 있게 함

## 6. 키 관리

Railway 환경변수에만 보관. `.env.example`만 리포에 커밋.

```
PUBLIC_DATA_API_KEY=
MOLIT_API_KEY=
KAKAO_REST_API_KEY=
KAKAO_JS_KEY=
KAKAO_NATIVE_APP_KEY=
ODSAY_API_KEY=
VWORLD_API_KEY=
GOOGLE_OAUTH_CLIENT_ID=
GOOGLE_OAUTH_CLIENT_SECRET=
KAKAO_OAUTH_CLIENT_ID=
KAKAO_OAUTH_CLIENT_SECRET=
```
